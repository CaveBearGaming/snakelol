# Snake Game Deluxe

A colorful, retro-style Snake game built with HTML, CSS, and JavaScript. Eat apples, grow your snake, and try to beat your high score!

## Features

- Classic snake gameplay with smooth grid-based movement
- Colorful snake with a gradient effect
- Apple collision and score tracking
- Increasing speed as you collect more apples
- Reset on wall or self collision
- Responsive canvas that works on desktop browsers

## Demo

You can play the game by opening `snake.html` in your browser.
